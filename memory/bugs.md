# bugs.md — Notable bugs and lessons

> Durable lessons from bugs, not a bug tracker. For each: what broke, root
> cause, the fix, and the lesson to carry forward. Newest at top.

> No implementation yet → no bugs yet. This template is here so the first real
> bug has a home.

## Template

```
## [Short title] — <date>
- **Symptom:** what was observed.
- **Root cause:** why it actually happened.
- **Fix:** what we changed.
- **Lesson:** what to remember / which invariant it touched.
- **Invariant/area:** e.g., Storage / Indexer / wikilink resolution.
```
